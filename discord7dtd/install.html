Discord integration for your 7 Days to Die server

Home How to Install GitHub

This is a bot for 7 Days to Die. It integrates your server's chat with Discord, allowing players to access to the game's chat from any device or location.

It works by connecting to your server's admin console and monitoring it for chat messages. Messages are then relayed to and from the server using Discord, allowing for communication between in-game players and Discord users.

Compatible with Windows, Linux, and Android systems. Powered by discord.js.
