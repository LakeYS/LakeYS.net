TriviaBot brings multiplayer trivia games to your Discord server! With over 3,000 questions and 24 categories, TriviaBot offers endless trivia fun. Categories include Film, Television, Anime & Manga, Sports, Science & Nature, and more.

How to Use

To use the bot, simply add it to your server and type trivia help in a text channel.

Type trivia play to start a trivia game and trivia categories to see a list of categories.

Development and Support

TriviaBot is powered by discord.js and the Open Trivia Database (OpenTDB).

For support, questions, and comments, you can join the Discord server or contact me directly at contact@LakeYS.net. You can also submit an issue on GitHub for bugs and support.
